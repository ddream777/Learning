### URL

##### URL格式

协议 + // + [用户名 + 密码] + Web服务器名 + [端口号] + 文件路径

##### 省略文件名的情况

+ http://www.test.com/dir/

实际会访问 /dir/index.html

+ http://www.test.com/

实际会访问 /index.html

+ http://www.test.com

同上

+ http://www.test.com/test

如果服务器上又test文件，则将其当作文件，否则当作目录
